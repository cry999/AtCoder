## D

:o:

まず過剰分をかぞえあげる。過剰分が偶数個なら過剰な数が `a` と `b` だとすると、 `aab` などと選択することで、`a` と `b` の過剰分を打ち消せる。奇数個なら過剰分が残り 1 個になるまで偶数個の処理を行い、あとは適当な数と組み合わせて消すので 1 つ余計に消す。`O(N)`
