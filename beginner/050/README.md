## C

:o:

絶対値が `a` のとき、その人がいたであろう位置は `(N-a-1)//2` or `N - (N-a-1)//2` の 2 通り。したがって、各位置に候補者が二人だけいる場合が正しい。ただし `N` が奇数の場合は中央は一人でないといけない。報告が正しい場合は `2 ^ (N//2)` 通りの正解が存在する。`O(N)`
