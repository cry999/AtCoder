## B

## C

:o:

## D

:o:

0 から n までの xor を Sn と表すと、

```math
Sn = n   (n % 4 == 0)
Sn = 1   (n % 4 == 1)
Sn = n+1 (n % 4 == 2)
Sn = 0   (n % 4 == 3)
```

が成り立つ。また xor の逆元は xor であり、A から B までの xor は SB から S{A-1} を取り除いたものなので、結局

```math
SB ^ S{A-1}
```

が求める答え。O(1)
