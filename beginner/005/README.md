## C

:o:

もっとも古いものが売れるなら売る、売れないなら破棄を繰り返しながら順番に客を捌く。`O(N+M)`

## D

:o:

まず長方形の累積和を求める(`O(N^2)`)。その累積和を利用して、各面積に対する最大値を計算する(`O(N^4)`)。さらに、`P` と同じ面積を必ず使わないといけないのではなく `P` 以下の面積なら `OK` なので、各面積に対する最大値を `P` 以下の面積に対する最大値に変更する(`O(N)`)。

以上より `O(N^4)`
