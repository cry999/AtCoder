# AtCoder Beginner Contest 003

## A

:o:

`N` 面ダイスの出目の期待値は `(N+1)/2`。`O(1)`

## B

:o:

`S` `T` の各文字をそれぞれ `s` `t` と表わす。`s` と `t` が等しい時は OK。`s` と `t` が異なる時は、どちらかが `@` と等しくてもう一方が `'atcoder'` のどれかであれば OK。以上以外の場合は全て NG。`O(|S|)`
