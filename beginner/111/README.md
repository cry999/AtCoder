## C

:o:

偶数番号列・奇数番号列それぞれで頻度とって最高頻度のものに合わせる。それぞれの列の最高頻度の数字が等しいなら、2 番目に頻度が高いものの頻度を比較して高い方はそちらを使用する様にする。ソートで O(n log(n))
