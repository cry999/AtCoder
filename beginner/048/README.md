## C

:o:

`a[i] + a[i+1]` が `x` より大きい場合は `a[i]` と `a[i+1]` のどちらかに操作を行わないといけないが引く数は同じなのでどちらに操作を使用が操作回数は同じ。`i` 以下に置いてすでに条件が満たされるとするなら `a[i+1]` に操作をする方が以降の操作で有利であることは自明。したがって、優先的に `a[i+1]` に操作を行い、足りない場合に `a[i]` にも操作を行う。`O(N)`
