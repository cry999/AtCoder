## B

:o:

`(i, j)` と `(N-1-i, N-1-j)` が入れ替わる。ただし `0 <= i < N` `0 <= j < N//2`。`O(N-2)`

## C

:o:

`N//5` と `N%5` から一発で求められそうだったがめんどくさかったので、`N%30 = 0` の時 `123456` のならびになることから、`N%30` 回操作するようにした。`O(N%30)`
