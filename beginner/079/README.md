## D

:o:

各数字を頂点とし、`c` を隣接行列とするグラフを考えればよい。あとは dijkstra 法などを用いて各数字から 1 への最短距離をあらかじめ出しておけば総和をとって終わり。`O(HW)`。ただし、dijkstra にかかる計算量は固定なので無視する。
