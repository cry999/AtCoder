## C

:o:

左から見て `0 ~ l` までの範囲の宝石を入手する場合のスコアの累積和(`Sl`)と、右から見て `r ~ M` までの範囲の宝石を入手する場合のスコアの累積和(`Sr`)をもつ。宝石を一つでも入手してなければいいので、ある `i` について宝石 `i` を収集しない場合のスコア `Sl[i-1] + Sr[i+1]` の最大値を求める。`O(N + M)`
