## D

:x:

確率の問題に読み替えは難しい。問題文通り `dp[i][x] = i 回目の操作で現在の X の値が x の時の最終的な X の期待値` とすると

```math
dp[i][x] = dp[i-1][x] * (N-i) / (N+1-i) + dp[i-1][x] / (N+1-i)
```

と表せる。これを `i` の大きい順に計算する。すると答えは `dp[0][X] * N!` となる。ここで計算途中に少数が入るのを防ぐために `dp[N][x]` の値を `x * N!` に初期化しておくと良い。

`O(NX)`
